# Forge Frontiers

A combination of idle tycoon and MMORPG, Forge Frontiers is a Java Minecraft Server that builds a unique gameplay experience utilizing Spigot plugins and additional web integrations.

Gameplay Features
- Idling (Generators)
- Mining
- Farming
- Fighting
- Fishing (Maybe)
- Trading & Selling
- Website Integration
